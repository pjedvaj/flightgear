This aircraft is developed by Petar Jedvaj (pjedvaj) based on the earlier version developed by Tomáš Komárek (Tomaskom) for the FightGear flight simulator, in collaboration with other members of the FGUK community. 

For the latest version of this aircraft visit https://github.com/pjedvaj/ALCA

List of changes:

aircraft is packed as 'ALCA' and can be used alongside with the old 'L-159' addon

- new aircraft 3D model
- added pilot model
- 3 new liveries
- new ALS shaders (reflection, normal and dirt maps)
- new effects (tyre smoke, heat, ALS shadow)
- new and updated animations (landing gear and disintegration)
- added generic HUD
- cockpit mockup with basic generic instruments
